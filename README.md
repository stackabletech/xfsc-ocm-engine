# ocm-engine version 1

#### Dependencies

Node 12
Python 2.5.0 >= <3.0.0
pnpm

### Setup local

1. pnpm install
2. pnpm prisma:generate
3. pnpm /app/:build

app options: attestation, connection, principal, ssi

### Docker compose

1. Go to compose dir

2. docker-compose up
